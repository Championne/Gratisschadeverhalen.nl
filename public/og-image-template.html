Instructies om OG-image te maken:

  1. Open dit bestand in Chrome of Firefox
  2. Klik rechts op de blauwe afbeelding hierboven
  3. Kies "Inspecteren" of druk F12
  4. Klik rechts op het .og-container element
  5. Kies "Capture node screenshot" (Chrome) of "Screenshot Node" (Firefox)
  6. Sla op als og-image.jpg of og-image.png
  7. Plaats het bestand in de /public map

Alternatief: Gebruik een screenshot tool zoals Greenshot of Windows Snipping Tool om exact 1200x630px te selecteren.
